# residuos-cat-demo

Dashboard público de demostración del producto **leads de residuos B2B Cataluña**.

⚠️ **Repo privado**. Datos anonimizados (sin emails/teléfonos/NIF). Pensado para
deploy en Streamlit Community Cloud.

## Arrancar localmente

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Deploy en Streamlit Cloud

1. https://share.streamlit.io/
2. New app → from existing repo
3. Repository: `Habanacasta88/residuos-cat-demo`
4. Branch: `main`
5. Main file path: `streamlit_app.py`
6. Deploy

URL pública resultante: `https://residuos-cat-demo.streamlit.app/` (o la que asignes).
